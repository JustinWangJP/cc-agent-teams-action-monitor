# システム使用書（ユーザーガイド）

## 1. はじめに

### 1.1 本システムの概要

Agent Teams Dashboardは、Claude CodeのAgent Teams機能をリアルタイムに監視・管理するためのダッシュボードアプリケーションです。

### 1.2 想定ユーザー

- Claude Codeを使用してAgent Teamsを運用する開発者
- チームのタスク進捗を確認したいプロジェクトマネージャー

### 1.3 前提条件

- Claude Codeがインストールされていること
- Agent Teams機能を使用していること
- ~/.claude/ディレクトリが存在すること

---

## 2. システム要件

### 2.1 ハードウェア要件

- ポート8000（バックエンド）

## 3. インストール手順

### 3.1 バックエンド

```bash
# 1. プロジェクトディレクトリに移動
cd cc-agent-teams-action-monitor/backend

# 2. 依存関係をインストール
pip install -e ".[dev]"

# 3. インストール確認
python -c "from app.main import app; print('OK')"
```

### 3.2 フロントエンド

```bash
# 1. プロジェクトディレクトリに移動
cd cc-agent-teams-action-monitor/frontend

# 2. 依存関係をインストール
npm install

# 3. インストール確認
npm run build
```

---

## 4. 起動手順

### 4.1 バックエンド起動

```bash
# 開発モードで起動
cd backend
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

起動成功時の出力例:
```
INFO:     Uvicorn running on http://127.0.0.1:8000
File watcher started on /Users/xxx/.claude
```

### 4.2 フロントエンド起動

```bash
# 開発モードで起動
cd frontend
npm run dev
```

起動成功時の出力例:
```
VITE v5.0.0  ready in 500 ms
➜  Local:   http://localhost:5173/
```

### 4.3 アクセス方法

ブラウザで http://localhost:5173/ にアクセスしてください。

## 6. 操作方法

### 6.1 チームの選択

1. チームカードをクリック
2. 選択されたチームがハイライト表示
3. もう一度クリックで選択解除

### 6.2 タスクの確認

1. タスクはステータス別に表示
   - Pending: 灰色
   - In Progress: 青色
   - Completed: 緑色

### 6.3 WebSocket接続状態の確認

ヘッダー右側のインジケーターで確認:
- ● 緑色: Connected
- ● 黄色: Connecting
- ● 赤色: Disconnected

## 7. トラブルシューティング

### 7.1 よくある問題と解決方法

| 問題 | 原因 | 解決方法 |
|------|------|----------|
| チームが表示されない | ~/.claude/teams/が空 | Claude Codeでチームを作成 |
| WebSocket切断 | バックエンド停止 | バックエンドを再起動 |
| ページが読み込めない | フロントエンド未起動 | npm run devを実行 |
| API エラー | CORS設定不一致 | 設定を確認 |

### 7.2 エラーメッセージ一覧

| エラー | 説明 | 対処法 |
|--------|------|--------|
| 404 Not Found | リソースが存在しない | データを確認 |

## 8. FAQ

### Q1: チームを作成するにはどうすればいいですか？

A: Claude Codeで `/team` コマンドまたは `TeamCreate` ツールを使用してください。

### Q2: リアルタイム更新が動作しません

A: 以下を確認してください:
1. バックエンドが起動している
2. WebSocket接続状態が Connected
3. ポート8000がブロックされていない

### Q3: データはどこに保存されますか？

A: `~/.claude/` ディレクトリ以下のJSONファイルに保存されます。
