## 4. 既知の問題

### BUG-004: file-history-snapshot データ構造不一致

**状態**: ✅ 修正済
**優先度**: P2
**影響**: 軽微（テストデータとの互換性問題）

**問題内容**:
- 実データ構造：`entry.snapshot.trackedFileBackups`
- テストデータ構造：`entry.fileChanges`

**修正内容**:
```python
# 修正前
file_changes = entry.get("snapshot", {}).get("trackedFileBackups", {})

# 修正後
file_changes = entry.get("snapshot", {}).get("trackedFileBackups", {})
if not file_changes:
    file_changes = entry.get("fileChanges", {})
```
